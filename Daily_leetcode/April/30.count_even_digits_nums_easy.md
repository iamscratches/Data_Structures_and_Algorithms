## âœ… Optimized Solution
```java
class Solution {
    public int findNumbers(int[] nums) {
        int count = 0;
        for(int num : nums) {
            int digitCount = 0;
            // Count digits
            while(num > 0) {
                digitCount++;
                num /= 10;
            }
            // Check if even
            if((digitCount & 1) == 0) {
                count++;
            }
        }
        return count;
    }
}
```
