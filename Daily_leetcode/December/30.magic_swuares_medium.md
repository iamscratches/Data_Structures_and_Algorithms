You are given an `m x n` integer grid.

A **3 Ã— 3 magic square** is defined as a grid where:

1. All numbers are **distinct**
2. All numbers are in the range **[1, 9]**
3. The **sum of each row, each column, and both diagonals** is the same

Return the number of **3 Ã— 3 magic squares** inside the grid.

## ğŸ§  Intuition

Since magic squares are **always 3 Ã— 3**, we can:

* Slide a 3 Ã— 3 window across the grid
* For each window, validate:

  1. All numbers are from **1 to 9**
  2. No duplicates
  3. All row, column, and diagonal sums are equal

The grid is small enough to brute-validate each 3 Ã— 3 subgrid efficiently.

## âš™ï¸ Approach

### ğŸ” Step-by-Step

1. If grid dimensions < 3 â†’ return 0
2. Slide a 3 Ã— 3 window using `(i, j)` as top-left
3. For each window:

   * Use a `boolean[10]` to ensure uniqueness and range
   * Compute reference sum using the first row
   * Verify:

     * All rows
     * All columns
     * Both diagonals
4. Count valid magic squares

```java []
class Solution {
    public int numMagicSquaresInside(int[][] grid) {
        int r = grid.length, c = grid[0].length, ans = 0;
        if (r < 3 || c < 3) return 0;

        for (int i = 0; i + 2 < r; i++) {
            for (int j = 0; j + 2 < c; j++) {
                boolean[] used = new boolean[10];
                boolean ok = true;

                // Check values are 1-9 and unique
                for (int x = 0; x < 3 && ok; x++) {
                    for (int y = 0; y < 3; y++) {
                        int v = grid[i + x][j + y];
                        if (v < 1 || v > 9 || used[v]) {
                            ok = false;
                            break;
                        }
                        used[v] = true;
                    }
                }
                if (!ok) continue;

                int s = grid[i][j] + grid[i][j+1] + grid[i][j+2];

                // Check rows
                for (int x = 0; x < 3; x++)
                    if (grid[i+x][j] + grid[i+x][j+1] + grid[i+x][j+2] != s) ok = false;

                // Check columns
                for (int y = 0; y < 3; y++)
                    if (grid[i][j+y] + grid[i+1][j+y] + grid[i+2][j+y] != s) ok = false;

                // Check diagonals
                if (grid[i][j] + grid[i+1][j+1] + grid[i+2][j+2] != s) ok = false;
                if (grid[i][j+2] + grid[i+1][j+1] + grid[i+2][j] != s) ok = false;

                if (ok) ans++;
            }
        }
        return ans;
    }
}
```
