# 📌 Check if Binary String Has at Most One Segment of Ones

## 📜 Problem Statement

**Link:**  
[https://leetcode.com/problems/check-if-binary-string-has-at-most-one-segment-of-ones/description/?envType=daily-question&envId=2026-03-06](https://leetcode.com/problems/check-if-binary-string-has-at-most-one-segment-of-ones/description/?envType=daily-question&envId=2026-03-06)

Given a binary string `s`, return `true` if `s` contains **at most one contiguous segment** of `'1'`.

Otherwise, return `false`.

(A contiguous segment is a maximal sequence of consecutive `'1'`'s.)

**Example:**  
Input: s = `"1001"`  
Output: false

Input: s = `"110"`  
Output: true

---

# 🧠 Understanding the Problem

- You can have *at most* one "block" of consecutive '1's anywhere in the string.
- Valid examples: "11111", "0011100", "0", "1", "000"  
  Invalid: "101", "110011"
- The simplest check: as you move through the string, do you ever find a '1' *after* a '0' that followed a previous block of '1'?

---

# 💻 Your Solution

### Approach:

- Track number of contiguous one-blocks.
- Start with block found if string starts with '1'.
- For every character, if we see '1' after a '0', increment the block count.
- If count exceeds 1, answer is false.

### Code (Cleaned):

```java []
class Solution {
    public boolean checkOnesSegment(String s) {
        int foundContiguous = s.charAt(0) == '1' ? 1 : 0;
        for (int i = 1; i < s.length(); i++) {
            char prev = s.charAt(i - 1), cur = s.charAt(i);
            if (cur == '1' && prev == '0') {
                foundContiguous++;
            }
        }
        return foundContiguous <= 1;
    }
}
```

---

# ⏱️ Complexity Analysis

- Single pass: O(n)
- Space: O(1)
- **Optimal for the problem**

---

# 🔎 Can We Do Better?

- For conciseness, check if `"01"` exists in the string.
- If yes, it means a block of ones ended and another started.

### Alternative (Succinct):

```java []
class Solution {
    public boolean checkOnesSegment(String s) {
        return !s.contains("01");
    }
}
```

---

# 💡 Key Insight

- A binary string can only have a single run of '1's uninterrupted by zeros for this to be true.

---

# 🔥 Comparison

| Approach         | Time  | Space | Notes           |
|------------------|-------|-------|-----------------|
| Block counting   | O(n)  | O(1)  | Explicit logic  |
| Contains "01"    | O(n)  | O(1)  | Concise, clever |

# 🧠 Takeaway

- For segment/runs problems, run-detection can be done either with state variables or a substring search.
- Always look for both approaches for clarity or brevity depending on needs.
