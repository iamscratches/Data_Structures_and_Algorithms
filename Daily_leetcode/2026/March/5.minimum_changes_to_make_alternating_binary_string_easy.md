# 📌 Minimum Changes To Make Alternating Binary String

## 📜 Problem Statement

**Link:**  
[https://leetcode.com/problems/minimum-changes-to-make-alternating-binary-string/?envType=daily-question&envId=2026-03-05](https://leetcode.com/problems/minimum-changes-to-make-alternating-binary-string/?envType=daily-question&envId=2026-03-05)

You are given a binary string `s`.  
You can change any character in `s` to `'0'` or `'1'` as needed.  
Return the minimum number of operations to make `s` alternating.  
An alternating binary string is one where no two adjacent characters are equal (e.g. "0101..." or "1010...").

**Example:**  
Input: `s = "0100"`  
Output: `1`  
Explanation: Change the last '0' to '1', string becomes "0101", which is alternating.

---

# 🧠 Understanding the Problem

- Two valid alternating strings of length n:  
  1. Start with '0': "01010..."  
  2. Start with '1': "10101..."  
- For each position: compare `s[i]` to expected pattern; count mismatches for both patterns.
- Return the smaller of the two possible flip counts.

---

# 💻 Your Solution

### Approach:

1. Iterate over the string, keeping track of how many flips needed for both "start with 0" and "start with 1" patterns.
2. For each character:
   - If not matching expected value for that alternation/order, increment that pattern's flip counter.
3. Result is the minimum of the two counters.

### Code (Cleaned):

```java []
class Solution {
    public int minOperations(String s) {
        int evenCount = 0, oddCount = 0;
        boolean alternator = true;
        char ch;
        for (int i = 0; i < s.length(); i++) {
            ch = s.charAt(i);
            if (ch == '0') {
                if (!alternator)
                    evenCount++;
                else
                    oddCount++;
            }
            if (ch == '1') {
                if (!alternator)
                    oddCount++;
                else
                    evenCount++;
            }
            alternator = !alternator;
        }
        return Math.min(evenCount, oddCount);
    }
}
```

---

# ⏱️ Complexity Analysis

- Single pass over s: O(n)
- Constant space (just counters)
- **Optimal solution**

---

# 💡 Key Insight

- Always compare against both possible alternating patterns.
- Minimum flips corresponds to the minimal difference from either.

# 🧠 Takeaway

- For string alternation problems, always check both "start with 0" and "start with 1".
- A simple positional scan is usually the best solution.
