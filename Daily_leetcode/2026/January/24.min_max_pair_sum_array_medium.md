You are given an array of even length `nums`.
You must pair the elements in such a way that:

* Each element is in **exactly one** pair
* You compute the **pair sum** for each pair
* You want to **minimize the maximum pair sum**

Return the **smallest possible maximum pair sum** after optimally pairing all elements.

## ğŸ§  Intuition

To minimize the **maximum pair sum**, we should pair:

* The **smallest element** with the **largest**
* The **2nd smallest** with the **2nd largest**, and so on.

This pairing ensures the largest values are â€œtemperedâ€ by the smallest ones, reducing the highest sum possible.

This is a classic greedy strategy.

```java []
class Solution {
    public int minPairSum(int[] nums) {
        Arrays.sort(nums);
        int len = nums.length;
        int maxPair = nums[0] + nums[len-1];
        for(int i = 1; i < len/2; i++){
            maxPair = Math.max(maxPair, nums[i] + nums[len-1-i]);
        }
        return maxPair;
    }
}
```
